üöÄ Data Collection Control Phase 3
‚ñ∂Ô∏è Start Collection

Begin background data collection daemon. Runs 5-minute collection cycle: fetches from Finnhub API, calculates real indicators, caches results for instant decision engine access.
When to use: Start of trading day - begins collecting real market data
‚èπÔ∏è Stop Collection

Stop background data collection gracefully. Cache remains fresh for 5 minutes. Useful for preserving API calls at end of day.
When to use: End of trading day or if you need to pause collection
üîÑ Restart Collection

Stop and restart data collection. Useful if you want to refresh immediately or reset collection cycle.
When to use: If you suspect stale data or need immediate refresh
‚ûï Add Ticker

Add custom ticker to data collection watchlist. Collector will fetch this ticker every 5 minutes. Note: SPY and QQQ are always tracked and protected.
‚úì wingman, add ticker TSLA
‚úì wingman, add ticker AAPL
When to use: When you want to start tracking a new ticker for analysis
‚ûñ Remove Ticker

Remove custom ticker from watchlist. Will stop collecting data for this ticker (cache expires in 5 min). Cannot remove SPY/QQQ - they're protected.
‚úì wingman, remove ticker TSLA
When to use: Stop tracking a ticker you no longer want to analyze
