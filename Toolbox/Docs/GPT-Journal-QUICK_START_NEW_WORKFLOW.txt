═══════════════════════════════════════════════════════════════════
STEP 1: MORNING - UPLOAD TO CHATGPT
═══════════════════════════════════════════════════════════════════

1. Open ChatGPT (web or app)
2. Start new conversation
3. Upload file: Journal/Journal.md
4. Tell ChatGPT: "Today is [DATE], let's trade"

✅ ChatGPT will read the instructions at the top of the file
✅ ChatGPT knows what data to collect
✅ ChatGPT knows how to format the EOD entry

═══════════════════════════════════════════════════════════════════

- Market observations (volatility, key levels, news)

- Emotional state (FOMO, patience, discipline)

ChatGPT will ASK for required data:
1. Cash & Sweep Vehicle balance (from broker)
2. OVERALL P/L YTD (from broker)
3. Daily P&L
4. Positions held overnight (if any)

6. Upcoming catalysts

⚠️ ChatGPT will NOT generate the journal until you provide this data!

═══════════════════════════════════════════════════════════════════
TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

**Issue:** ChatGPT doesn't ask for account data
**Fix:** Make sure you uploaded Journal.md (not a different file)
       The instructions are at the top of that file

**Issue:** Workflow says "No balance data found"
**Fix:** Check that your journal entry has the Account Summary section
       with real numbers (not placeholders)

**Issue:** Portfolio recommendations use old balance
**Fix:** Make sure you saved the downloaded Journal.md file
       Then re-run the workflow

**Issue:** ChatGPT generates entry with $XX,XXX.XX placeholders
**Fix:** Tell ChatGPT to stop and ask you for the real numbers first
